# Arkhanths — Downloads

Official downloads for the [Arkhanths](https://arkhanths.com) client.

**[Get the latest version here](https://github.com/DevilTekGame/Arkhanths-Releases/releases/latest)**

## Which file do I need?

| System | File |
| --- | --- |
| Windows | `Arkhanths_x.x.x_x64-setup.exe` (recommended) or the `.msi` |
| macOS (Apple Silicon & Intel) | `Arkhanths_x.x.x_universal.dmg` |
| Ubuntu / Debian | `Arkhanths_x.x.x_amd64.deb` |
| Fedora / openSUSE | `Arkhanths-x.x.x-1.x86_64.rpm` |
| Any Linux distro | `Arkhanths_x.x.x_amd64.AppImage` (no install needed) |

## Install notes

- **Windows:** run the setup. WebView2 is installed automatically if missing.
- **macOS:** open the `.dmg` and drag Arkhanths to Applications. The first time, right-click the app and choose **Open** (the app is not yet notarized by Apple).
- **Ubuntu/Debian:** `sudo apt install ./Arkhanths_x.x.x_amd64.deb`
- **AppImage:** `chmod +x Arkhanths_x.x.x_amd64.AppImage`, then run it.
